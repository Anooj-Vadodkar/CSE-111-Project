FAQ | Questions, Concerns, And Comments

Why this topic?
We both really like video games, as they played pivotal parts in our lives growing up. With that in mind, we figured that it would be a fun project to make a sort of list of the history of video game sales, one that could be updated and saved in the internet. Using data from Kaggle, sqlite, and basic web development, we were able to make this website as an online, interactive look.
What tools were used?
HTML Template from Start Bootstrap for the website format, Kaggle for finding the database, sqlite for the database.
Why can't we see our entries immediately reflected?
There are two main reasons for this. The first, and probably more pressing one, is security. Direct entry into databases opens up the door to lots of unwanted activity like SQL Injections. Had this been the only issue, we would have handled it. However, there is a second reason: We need to make sure the games are okay to be put into the list. There are a lot of games out there that... probably wouldn't be good additions. The sheer amount of Early Access Open World Survival Crafting games out there would skyrocket this database, as well as not really being what we're looking for. It is because of those two reasons that we take in suggestions and then add them based on whether they fit.
